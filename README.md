# CRUD Perpustakaan Sederhana

Aplikasi sederhana untuk mengelola data buku di perpustakaan menggunakan **Laravel**.  
Pengguna dapat melakukan operasi CRUD (Create, Read, Update, Delete) untuk data buku.
